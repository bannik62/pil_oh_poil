Introduction

Ce projet utilise Svelte pour créer une One Page composée de 7 sections modulaires :

  • Header : Utilisé comme page principale pour le One Page.
  • Présentation : Petite présentation de l'entreprise.
  • Prestation : Affiche les tarifs et prestations.
  • Caroussel : Affichage du magasin en photo.
  • Section Géographique : Adresse avec une carte géographique.
  • Footer : Informations générales.

Structure des Modules

La structure du projet est organisée de manière modulaire :

src/
│
├── App.svelte
└── components/
    ├── Header.svelte
    ├── Presentation.svelte
    ├── Prestation.svelte
    ├── Caroussel.svelte
    ├── SectionGeo.svelte
    └── Footer.svelte

Header

Le Header sert de page principale pour le One Page :

// src/components/Header.svelte
<script>
    export let title = "Bienvenue sur Pile Oh Poil";
</script>

<header>
    <h1>{title}</h1>
    <nav>
        <a href="#presentation">Présentation</a>
        <a href="#prestation">Prestation</a>
        <a href="#caroussel">Photos</a>
        <a href="#geo">Contact</a>
    </nav>
</header>

Présentation

La section Présentation donne un aperçu de l'entreprise :

// src/components/Presentation.svelte

Conclusion

Cette structure modulaire avec Svelte permet une maintenance et une évolution aisées du site. Chaque section étant encapsulée dans un module, le développement reste organisé et cohérent.
